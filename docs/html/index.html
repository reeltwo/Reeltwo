ReelTwo
RSeries astromech firmware

This is the API documentation for Reeltwo. Reeltwo is an Arduino class library to simply building RSeries Astromech droids. You can download it as a tgz archive for offline reading.

If this is your first contact with Reeltwo, have a look at the getting started page that show you how to write and compile your first sketch using Reeltwo.

The quick reference pages give you a quite complete description of the API in a very condensed format that is specially useful to recall the syntax of a particular feature, or to have a quick look at the API. They currently cover the two following feature sets, and more will come in the future:

Dome

  • Bad Motivator
  • Dome Button Controller
  • Holoprojector
  • Holoprojector OLED Display
  • RSeries LogicEngine logics
  • Teeces logics
  • Magic Panel
  • Teeces PSI

Body

  • Charge Bay Indicator
  • Data Panel

Servo Control

  • Servo Dispatch PCA9685
  • Servo Dispatch Direct PWM
  • Servo Sequencer

You can also use the search box to quickly get to the documentation of a given class or function.
